# MemoryGuard Docs

MemoryGuard keeps AI coding-agent context files current and secret-safe.

Start with the root README, then try:

```bash
memoryguard init
memoryguard remember "This project uses Flask for the backend."
memoryguard sync
memoryguard demo
```

Core ideas:

- Local-first project memory.
- Pending review for transcript-captured facts.
- Secret-safe generated context files.
- No required external LLM API.
