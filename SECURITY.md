# Security Policy

MemoryGuard is local-first alpha software. Please do not open public issues for vulnerabilities or accidental secret exposure bugs.

## Reporting

Use GitHub private vulnerability reporting if available, or contact the maintainers privately.

## Secret Safety

MemoryGuard redacts common secret-looking values before displaying capture candidates, storing approved memories, or rendering generated context. Redaction is best-effort. Always inspect generated context before sharing it.

## Supported Versions

The public alpha is pre-1.0. Security fixes will target the current main branch.
